Leonis Terraform

Programming Language: GDScript | Software: Godot 4.4

Leonis Terraform is a simple terrain generation tool made in Godot.
It aims to provide a simple yet powerful solution for creating terrain and heightmaps.

An example terrain made in Terraform's prototype build.

This tool will also be available as a Godot plugin, allowing you to create Terrain nodes and configure it however you want.

Current Features:

  • Node-based system for creating terrain cells
  • Graph editor for configuring nodes
  • Interactive viewport
  • Export as a heightmap texture or Godot resource

Planned Features:

  • Export as 3D model (.obj, .dae etc)
  • Managing scene properties (light direction, colour)
  • Configurable cell size
